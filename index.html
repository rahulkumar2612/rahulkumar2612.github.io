Fun With Candlesticks

My first web-app! This is a virtual trading app created using Python based micro framework Flask. Registered users get virtual money ($10k to start with) Users can buy stock, build their portfolio and sell stock to cash in on their profits, getting a chance to test their trading strategies before investing real money. History of transactions is provided along with a leaderboard position. Live market data is provided by integrating a 3rd party api. Site is currently hosted on pythonanywhere.com using WSGI server. Tech Stack: Flask, Python, SQL, JavaScript, HTML/CSS, Bootstrap.
